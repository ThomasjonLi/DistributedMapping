# DistributedMapping

## Requirements
* [Eigen](http://eigen.tuxfamily.org)
* [g2o](https://github.com/RainerKuemmerle/g2o)
* [Sophus](https://github.com/strasdat/Sophus)
* [evo](https://github.com/MichaelGrupp/evo)(optional)

## Compilation
```
mkdir build
cd build
cmake ..
make -j4
./kitti(or ./sphere)
```
